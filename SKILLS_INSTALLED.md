# Local Codex Skills Installed

Installed for this project in `.codex/skills`:

- `platform/senior-frontend`: React/TypeScript/Tailwind frontend implementation guidance.
- `platform/react-best-practices`: React performance, localStorage schema, rendering and state rules.
- `quality/webapp-testing`: local web app interaction testing patterns and Playwright helpers.
- `design/accessibility-auditor`: accessibility and older-user-friendly UI checks.
- `quality/test-driven-development`: focused test planning and anti-pattern checks.
- `quality/code-reviewer`: structured code review checklist and diff review helper.

Suggested usage for this project:

1. Use `senior-frontend` and `react-best-practices` during implementation.
2. Use `webapp-testing` after the first runnable version to verify tab switching, check-in state changes, inventory deduction, reset demo data, and assistant output.
3. Use `accessibility-auditor` before final polish for button size, contrast, labels, and mobile friendliness.
4. Use `code-reviewer` plus the already-installed global `adversarial-review` before deployment.

Note: Restart Codex if these local skills do not appear automatically in a new session.
